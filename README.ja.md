# koshi-school

福井県立高志中学校の「高志学」プロジェクトを、WebブラウザとVRデバイスで体験できるWebアプリです。

## デモ
https://code4fukui.github.io/koshi-school/

https://fukuno.jig.jp/3509

## 機能
- 3Dモデルを使った中学校の校舎の探索
- 教室や施設の写真表示
- VRデバイス(Oculus Touch)での操作と移動

## 必要環境
- WebブラウザまたはVRデバイス(Oculus Touch)
- WebGLに対応したデバイス

## 使い方
Webブラウザで上記のデモリンクにアクセスするか、VRデバイスを使ってアプリを起動してください。

## データ・API
本アプリは福井県立高志中学校の「高志学」プロジェクトのデータを利用しています。
