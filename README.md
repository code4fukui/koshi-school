# koshi-school

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A web application that allows users to explore a 3D virtual tour of Koshi High School and view a slideshow of images from the "Koshi Gaku" project.

## Demo
https://code4fukui.github.io/koshi-school/
https://fukuno.jig.jp/3509

## Features
- 3D virtual tour of Koshi High School
- Viewer with camera controls and teleportation
- Automatic slideshow of images on a screen in the scene

## Requirements
- A-Frame compatible web browser

## Usage
Load the `index.html` file in a web browser to view the 3D scene.

## Data / API
This application uses data from the "Koshi Gaku" project of Fukui Prefectural Koshi High School.
